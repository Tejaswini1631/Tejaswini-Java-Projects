# Steps to Execute the Program

1. Download MySQL Connector/J

2. Extract the ZIP file

3. Copy the .jar file path

4. Open Command Prompt

5. Go to project source folder

cd C:\Java_Projects\StudentManagementSystem\src

6. Compile the Java files

javac -cp ".;JAR_FILE_PATH" com\app\*.java

7. Run the program

java -cp ".;JAR_FILE_PATH" com.app.Main

8. Enter menu options to perform operations
