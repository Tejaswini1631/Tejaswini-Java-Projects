# 📊 Student Grade Tracker

## 📌 Description
A Java console-based application to track student marks and generate reports like average, highest, and lowest scores.

## 🚀 Features
- Add Student (Name & Marks)
- View Student Report
- Calculate Average Marks
- Display Highest & Lowest Marks

## 🛠️ Technologies
- Java
- ArrayList
- OOP Concepts

## 📂 Files
- MainApp.java

## ▶️ How to Run
1. Open project in IDE (VS Code / Eclipse)
2. Run `MainApp.java`
3. Use menu options:
   - Add Student
   - View Report
   - Exit

- Stores student data using ArrayList :contentReference[oaicite:0]{index=0}  
- Calculates average, highest, and lowest marks dynamically
